1. ATIVIDADES

1.1. Atividade

A seção Atividade constitui o núcleo operacional do Sistema de Gerenciamento de Demandas do Arena GO. É por meio desta funcionalidade que são realizadas a consulta, o acompanhamento, o controle e o cadastro dos chamados de manutenção vinculados ao contrato do TJSP.

Nesta tela, o usuário poderá:

  • Monitorar o status dos chamados;
  • Aplicar filtros avançados de consulta;
  • Analisar indicadores operacionais;
  • Exportar informações;
  • Cadastrar novas atividades.

Esta funcionalidade é utilizada tanto para a gestão operacional diária quanto para fins de controle contratual, auditorias e acompanhamento de desempenho.

1.1.1. Visão Geral do Dashboard

O Dashboard de Atividades é composto pelos seguintes elementos:

  • Painel de filtros de consulta;
  • Indicadores resumidos de status das atividades;
  • Lista detalhada dos chamados cadastrados;
  • Botões de ação rápida.

1.1.2. Filtros de Consulta

Os filtros permitem refinar a pesquisa dos chamados de acordo com critérios operacionais, contratuais e temporais.

ID

Tipo: Campo digitável

Função: Utilizado para informar o número interno EQS do chamado (não corresponde ao ID do cliente).

Situação

Tipo: Lista suspensa (múltipla escolha)

Opções:

  • Aguardando confirmação
  • Aberto
  • Em andamento
  • Fechado
  • Fechado com pendência
  • Cancelado

Observação: Por padrão, o sistema mantém quatro situações previamente selecionadas.

Contrato

Tipo: Lista suspensa (múltipla escolha)

Opção padrão: TJSP

Tipo

Tipo: Lista suspensa (múltipla escolha)

Opções:

  • Abastecimento
  • Acompanhamento
  • Adequação
  • Comunicado
  • Contratação
  • Corretiva
  • Corretiva programada
  • Desinstalação
  • GMG
  • Implantação
  • Inspeção
  • Instalação
  • Preditiva
  • Preventiva
  • Preventiva de Elétrica
  • Preventiva de Hidráulica
  • Preventiva Predial
  • Registro
  • Retirada de pendência
  • Teste
  • Vistoria
Atividade

Tipo: Lista suspensa (múltipla escolha)

Opções:

  • Alvenaria
  • Caça Vazamento
  • Diário
  • Elétrica
  • Hidráulica
  • Pintura
  • Poda
  • Reparo/Instalação Geral
  • Telhado
  • Entre outros
Estado

Tipo: Lista suspensa (múltipla escolha)

Função: Utilizada para filtro por estado

Macroárea

Tipo: Lista suspensa (múltipla escolha)

Função: Região macro (não aplicável)

Local

Tipo: Campo com autocompletar

Função: Permite selecionar polos e prédios cadastrados

Ativo do Local

Tipo: Campo com autocompletar

Função: Sem função operacional no momento

LPU

Tipo: Lista suspensa

Opções:

  • Todos
  • Com LPU
  • Sem LPU (padrão)
Responsável

Tipo: Campo com autocompletar

Função: Permite selecionar colaboradores EQS

Ocorrência

Tipo: Período (intervalo de datas)

Função: Refere-se à data de abertura do chamado pelo cliente

Prazo

Tipo: Período (intervalo de datas)

Função: Refere-se à data de agendamento para execução da atividade

Fechamento

Tipo: Período (intervalo de datas)

Função: Refere-se à data de conclusão da atividade

Vencimento

Tipo: Lista suspensa

Opções:

  • Todos
  • Vencido
  • Em 5 dias
  • Em 15 dias
Cód. Cliente

Tipo: Campo digitável

Função: Aceita exclusivamente o ID completo do chamado aberto pelo cliente

Prioridade

Tipo: Lista suspensa (múltipla escolha)

Função: Utilizada para classificação do nível de prioridade

Avaliação

Tipo: Lista suspensa (múltipla escolha)

Função: Utilizada para classificação da avaliação do atendimento

1.1.2.1. Regras e Restrições de Consulta

Para consultas com períodos extensos, aplicam-se as seguintes regras:

  • É obrigatório informar Contrato ou Responsável;
  • Caso contrário, o período máximo permitido para consulta é de 6 meses;
  • O descumprimento das regras gera alerta automático em tela.

1.1.3. Botões de Ação (Tela)

Localizados na parte inferior do painel de filtros:

  • Consultar: executa a pesquisa conforme os filtros aplicados;
  • Limpar: restaurar os filtros padrão do sistema;
  • Exportar Excel: exporta a lista de atividades conforme os filtros utilizados;
  • Favoritos: permite gerenciar pesquisas salvas, colunas exibidas e indicadores. Ao clicar na seta, é possível editar o dashboard.

1.1.4. Indicadores de Atividades

Os indicadores funcionam como filtros rápidos.

Funcionamento:

Ao selecionar um indicador, o sistema aplica automaticamente o filtro correspondente e atualiza a lista de atividades. Esses indicadores podem ser personalizados por meio da função Favoritos.

Indicadores disponíveis:
  • Crítico
  • Vencido
  • Aguardando
  • Abertos
  • Andamento
  • Pendentes
  • Fechado

1.1.5. Lista de Atividades

Exibe os chamados conforme os filtros aplicados, com colunas como ID, Tipo, Local, Responsável, Situação, entre outras. As colunas exibidas podem ser personalizadas nos favoritos.

1.1.5.1. Botões de Ação por Linha

Disponíveis na coluna Ação de cada registro:

  • Editar: permite alterar dados do chamado;
  • Gerar PDF de Evidências: gera relatório com fotos e anexos;
  • Excluir/Cancelar: cancela ou remove o chamado, conforme as permissões do usuário.

1.1.6. Botão Novo

O botão Novo, localizado no topo do dashboard em Atividades > Atividade, permite registrar um novo chamado e redireciona o usuário para a página de Nova Atividade.

1.1.6.1. Nova Atividade

O formulário de Nova Atividade é utilizado para o registro inicial do chamado.

Local de Atendimento*

Obrigatório: Sim

Como preencher:

  • Digite parte do nome do local, contrato ou endereço;
  • Selecione a opção correta na lista apresentada.

Observação: Somente após a seleção do Local de Atendimento os demais campos do formulário serão habilitados. Caso o local não exista, utilize o link para cadastro de novo local.

Atividade*

Tipo: Lista suspensa obrigatória

Data Ocorrência*

Função: Data de abertura do chamado pelo cliente. Após o preenchimento, o campo Prazo é habilitado.

Prazo*

Função: Data de agendamento para execução da atividade

Previsão de Chegada ao Local de Atendimento

Tipo: Campo opcional de data e hora

Previsão de Retorno de Energia

Tipo: Campo opcional, aplicável apenas a ocorrências elétricas

Atendimento Crítico

Tipo: Checkbox opcional, com padrão "Não"

Foto Ocorrência

Tipo: Upload opcional de imagens relacionadas à ocorrência

Responsável*

Função: Colaborador responsável pela supervisão da atividade

Observação p/ Responsável

Função: Campo destinado à descrição dos chamados repassados pelo cliente

Cód. Interno do Cliente

Tipo: Campo para o ID do chamado do cliente

Observação Cliente

Função: Campo opcional para orientações internas. Não é possível alterar após o salvamento.

Fluxo resumido de cadastro:

  1. Selecionar o Local de Atendimento;
  2. Preencher a Data de Ocorrência;
  3. Selecionar a Atividade e o Responsável;
  4. Preencher os campos opcionais, se necessário;
  5. Clicar em Salvar.

Observação: Após o salvamento, a atividade é registrada com a situação Aberto e o sistema redireciona para a tela de Cadastro de Atividade.

1.1.6.2. Cadastro de Atividade (Nova Atividade)

Botões do topo:

  • Abrir Orçamento: abre nova guia para inclusão de orçamento;
  • Exportar PDF: gera relatório com fotos e anexos;
  • Bloquear: bloqueia o acesso do cliente ao relatório.

Abas disponíveis:

Atividade

Permite alterar informações como:

  • Situação
  • Data de Ocorrência
  • Prazo
  • Previsão de Chegada
  • Previsão de Retorno de Energia
  • Atendimento Crítico
  • Responsável
  • Observação p/ Responsável
  • Cód. Interno do Cliente
  • Foto da Ocorrência

Inclui também o link Atividade no Local, que lista os serviços executados naquele local.

O histórico apresenta todas as interações realizadas na atividade, enquanto os comentários são utilizados para mensagens internas de controle.

Relatório

Contém as informações detalhadas do chamado:

  • Descrição do Chamado
  • Condição Antes
  • Condição Durante
  • Condição Depois
  • Conclusão do Chamado
Cadastro de Interações

Utilizado para inserir:

  • Descrição do chamado do cliente
  • Fotos do antes, durante e depois
  • Conclusão detalhada do serviço executado

Observação: Para carregar imagens, selecione a foto e clique em enviar após o carregamento.

LPU

Status: As funções desta aba não são utilizadas neste momento.

1.1.6.3. Cadastro de Atividade (Edição de Atividade em Aberto)

Durante a edição da atividade, é possível alterar:

Na aba Atividade:
  • Data de Ocorrência
  • Prazo
  • Previsão de Chegada ao Local de Atendimento
  • Previsão de Retorno de Energia
  • Atendimento Crítico
  • Responsável
  • Observação p/ Responsável
  • Cód. Interno do Cliente
Na aba Cadastro de Interações:
  • Descrição do Chamado
  • Condição Antes
  • Condição Durante
  • Condição Depois
  • Conclusão do Chamado

Observação importante: Ao salvar a atividade com a situação fechado, não será possível realizar alterações, exceto em situações específicas.

1.1.6.4. Cadastro de Atividade (Edição de Atividade Fechada)

Ao abrir a edição de uma atividade fechada, são disponibilizados os seguintes botões:

  • Enviar p/ aprovação do cliente: envia e-mail com o PDF do relatório (funções Aprovar/Reprovar não funcionais no momento);
  • Fechar com pendência: utilizado quando há necessidade de aguardar compra ou chegada de material;
  • Nova atividade relacionada: cria um novo chamado vinculado à atividade atual.
Edições possíveis:
  • Alterar a situação de Fechado para Fechado com pendência, liberando a edição;
  • Alterar o Cód. Interno do Cliente (ID do chamado do cliente).
